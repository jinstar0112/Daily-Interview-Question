## 今日面试题
2019-07-19

> 第 109 题：编程题
>
> ```js
> var entry = {
>   a: {
>     b: {
>       c: {
>         dd: 'abcdd'
>       }
>     },
>     d: {
>       xx: 'adxx'
>     },
>     e: 'ae'
>   }
> }
> 写个程序把entry转换成如下对象
> var output = {
>   'a.b.c.dd': 'abcdd',
>   'a.d.xx': 'adxx',
>   'a.e': 'ae'
> }
> ```
